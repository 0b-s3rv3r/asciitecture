A simple text-based graphics game library for zig . Currently, it only supports Linux. 

### Dependencies
- zig 0.13.0
- libx11-dev
- libxrandr-dev

e.g.:
```sudo apt install libx11-dev libxrandr-dev```

### Run example
```zig build example```

### Run tests
```zig build test```
